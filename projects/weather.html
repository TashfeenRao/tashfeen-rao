Calculator

Implemented asynchronous Javascript, Used open weather API service to get live weather information. Users can get any city temperature in both scales Centigrade or Fahrenheit by typing city names. Built with: HTML, CSS, Javascript, ES6, Webpack, Babel, Jest, Eslint, GitHub.

Key Features

  • User can type city name for weather information
  • User can get the temperature in both scales in both scales oC or oF

How To Uses

  • git clone https://github.com/TashfeenRao/weather-app
  • cd weather-app
  • npm install
  • npm start
